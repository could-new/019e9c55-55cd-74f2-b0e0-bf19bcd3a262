# Home Tuitions

A Flutter application for finding and booking home tutors.

## Features
- Browse available tutors with their subjects and hourly rates.
- View detailed tutor profiles.
- Book tutoring sessions.
- Manage your bookings.

## Tech Stack
- Flutter
- Material Design 3

## Setup
1. Run `flutter pub get`
2. Run `flutter run`
